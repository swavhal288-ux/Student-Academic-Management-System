# Student Academic Management System

## 📌 Project Overview
The Student Academic Management System is an academic Database Management System (DBMS) project developed using SQL.  
It is designed to efficiently manage student academic records such as storing, updating, and retrieving student information.

---

## 🎯 Objective
To understand and implement database concepts for managing student academic data in an organized and efficient manner.

---

## 🛠️ Technologies Used
- SQL (Structured Query Language)
- DBMS Concepts
- Microsoft PowerPoint (for presentation)
- Microsoft Word (for report documentation)

---

## 📁 Project Contents
- 📊 Project Presentation (PPT)
- 📄 Project Report (PDF)
- 🧾 SQL Implementation Screenshots (included in PPT and Report)

---

## ⚙️ Implementation Details
- Created and managed student database using SQL
- Performed operations like:
  - Insert records
  - Update records
  - Retrieve data
  - Delete records
- All SQL queries and outputs are documented in the PPT and Report
